# Trendy Flutter CRUD

A trendy Flutter CRUD application showcasing **Bento Grid** design and **Riverpod Generator (Annotation)**.

## Features
- **Bento Grid Layout**: Modern card-based UI.
- **Riverpod Annotation**: Uses the latest `@riverpod` code generation style.
- **Material 3**: Clean and minimal design.
- **Full CRUD**: Create, Read, Update, and Delete posts.

## Tech Stack
- Flutter
- `flutter_riverpod` + `riverpod_generator`
- Material 3

## Getting Started

1. Clone the repo
2. Run `flutter pub get`
3. Generate code: `dart run build_runner build`
4. Run the app: `flutter run`
